Welcome to the main page of the "Sentra" Telegram bot organization.

Sentra is a powerful chatbot for managing, administering, and managing chats, similar to Rose and Group Help. While it currently lacks functionality compared to these tools, we plan to expand it significantly and add killer features you'll definitely love. The bot is completely free, with no ads, no donation features, no subscriptions, and no restrictions on donations whatsoever.

Sentra is all about simplicity and convenience, without a cluttered interface, confusing commands, or any other confusion. We regularly optimize the code, add clear descriptions to each command, and will soon create a comprehensive FAQ covering all the bot's features. Stay tuned for updates so you don't miss out.

By using the bot, you acknowledge that you have read and accepted the [bot user agreement](https://teletype.in/@thesentra/SUA). This is a small stipulation, but it's necessary for both us and you. If you violate the terms of the user agreement, we reserve the right to restrict your access to our bot. The user agreement is closely linked to the [Telegram Temps of Service](https://telegram.org/tos/ru), so we recommend that you also familiarize yourself with it.

**The bot is currently under active development and refinement!** Anything added during development may not make it into the final version of the bot. Please be understanding. If you find any bugs or flaws, or if you'd like to suggest an idea to improve the bot experience, please contact one of the lead developers: [@podyshka1](https://t.me/podyshka1), [@cursenota](https://t.me/cursenota), [@shuseks](https://t.me/shuseks), [@n1ksly](https://t.me/n1ksly).

The bot officially supports two languages: __``üá∑üá∫ Russian``__ and __``üá∫üá∏ English``__. You can change the language at any time. All official **Sentra** resources also natively support these two languages.
